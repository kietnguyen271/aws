Worklog của em được ghi lại trong suốt 12 tuần thực tập tại First Cloud Journey, bao gồm các nội dung chính như sau:

**Tuần 1:** [Làm quen với môi trường AWS, IAM, MFA, AWS CLI và các dịch vụ cơ bản](1.1-week1/)

**Tuần 2:** [Mạng AWS: VPC, Subnet, Route Table, VPN, VPC Peering và Transit Gateway](1.2-week2/)

**Tuần 3:** [Làm việc với S3, RDS, Cloud9; tham gia Cloud Day Vietnam; nghiên cứu cơ bản cho kiến trúc dự án nhóm](1.3-week3/)

**Tuần 4:** [Tìm hiểu nhóm dịch vụ Storage, VM Import/Export và bắt đầu xây dựng proposal dự án](1.4-week4/)

**Tuần 5:** [Thiết kế giao diện dự án (Admin/Staff), xây dựng login flow và phối hợp với Backend](1.5-week5/)

**Tuần 6:** [Thiết kế kiến trúc AWS cho dự án, nhận phản hồi mentor và chỉnh sửa; chuẩn bị thi giữa kỳ](1.6-week6/)

**Tuần 7:** [Triển khai Frontend serverless, API Gateway, Lambda và thực hành xác thực với Cognito](1.7-week7/)

**Tuần 8:** [Nghiên cứu các trụ cột AWS Architecture: Security, Resilience, Performance, Cost Optimization](1.8-week8/)

**Tuần 9:** [API Gateway nâng cao: Path Parameters, Query Strings, Integration Proxy, Usage Plan và CloudWatch Logs](1.9-week9/)

**Tuần 10:** [Xây dựng Lambda CRUD, import Excel vào DynamoDB, kiểm thử API bằng Postman và triển khai validation](1.10-week10/)

**Tuần 11:** [DevOps trên AWS: IaC, Containers, Canary Deployment, Stage Variables, Route 53](1.11-week11/)

**Tuần 12:** [CloudFront, Docker, ECR, phát triển Auto Scoring Lambda và lưu kết quả vào DynamoDB](1.12-week12/)
